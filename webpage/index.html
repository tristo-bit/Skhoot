Features

Everything you need to stay organized

Powerful AI-driven tools designed to help you find, manage, and organize your digital life.

Intelligent File Search

Multi-engine search with fuzzy matching, CLI integration, and AI-powered suggestions. Find any file in milliseconds with our Rust-based search engine.

  • Hybrid search combining multiple engines
  • Content search with snippet extraction
  • Real-time performance metrics

Voice Interface

Speak naturally to search, command, and interact. Real-time transcription with audio visualization.

AI Chat

Powered by Google Gemini for natural conversations. Get help with files, tasks, and more.

Disk Analysis

Visualize disk usage, identify large files, and get smart cleanup recommendations.

Modern Design

Embossed glassmorphic UI with light and dark themes. Smooth animations and accessibility built-in.
